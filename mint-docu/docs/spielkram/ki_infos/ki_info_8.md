Ja, die Kombination aus Python und Open-Source-Spielen ist ideal, um selbst kreativ zu werden, da die Sprache leicht zu lernen ist. Hier sind die besten Projekte, die entweder komplett in Python geschrieben sind oder Python für das Modding nutzen:
1. Unknown Horizons (Komplett in Python)
Dies ist wohl das bekannteste Open-Source-Projekt für friedliche Strategie-Fans, das fast vollständig in Python (und XML) entwickelt wurde.

* Genre: Aufbau- und Handelsstrategie (sehr ähnlich zu Anno).
* Fokus: Siedlungsbau, Ressourcenmanagement und Diplomatie. Es ist friedlich und für 2 bis 4 Spieler im Multiplayer ausgelegt.
* Technik: Da es 2D-Grafik nutzt, läuft es perfekt auf Systemen ohne dedizierte Grafikkarte.
* Modding: Da der gesamte Code in Python vorliegt, können Sie fast jeden Aspekt des Spiels direkt im Quellcode verändern oder erweitern.

2. Civilization IV (Modding-König mit Python)
Obwohl das Spiel selbst nicht Open Source ist, ist es mittlerweile oft sehr günstig zu finden und ein Paradebeispiel für Python-Modding.

* Technik: Die gesamte Spiellogik und das Interface werden über Python-Skripte gesteuert.
* Modding: Die Community hat riesige "Total Conversions" (wie Fall from Heaven) nur mit Python und XML erschaffen. Es ist ideal, um zu lernen, wie man Spielmechaniken per Skript beeinflusst.

3. Pygame-Projekte (Für eigene kleine Spiele)
Wenn Sie selbst etwas Kleines erschaffen wollen, ist Pygame die Standard-Bibliothek. Es gibt zahlreiche Open-Source-Vorlagen auf GitHub:

* Beispiel: Suchen Sie nach "Multiplayer Pygame" Projekten. Diese sind oft minimalistisch (ähnlich wie Snake oder einfache Handelsspiele), aber man kann den Code sofort verstehen und anpassen.

4. Frets on Fire (Musik & Geschicklichkeit)
Ein lustiges, kostenloses Spiel im Stil von Guitar Hero.

* Technik: Es ist komplett in Python geschrieben.
* Kreativität: Man kann eigene Songs hinzufügen oder die Spielmechanik durch Python-Skripte anpassen. Es ist zwar kein Handelssspiel, aber ein tolles Beispiel für ein performantes Python-Spiel.

5. Battlefield 2 (Klassiker mit Python-Backend)
Falls Sie doch einmal einen Shooter mit Bots ausprobieren wollen: Die gesamte Server-Logik und die Punkteberechnung von Battlefield 2 laufen über Python. Viele Modder haben hier gelernt, wie man Multiplayer-Events programmiert.
Warum Python für Ihre Anforderungen?
Python-Spiele sind oft CPU-gebunden und nicht grafikintensiv, was perfekt zu Ihrem Wunsch nach Spielen ohne Grafikkarte passt. Besonders Unknown Horizons bietet Ihnen genau die Handels- und Aufbaumechanik, nach der Sie suchen, kombiniert mit der Möglichkeit, tief in den Python-Code einzutauchen.
Mein Tipp: Laden Sie sich Unknown Horizons von GitHub oder der offiziellen Webseite herunter. Schauen Sie in den Ordner horizons/world/, um zu sehen, wie die Gebäude und Warenwirtschaft in Python definiert sind.
