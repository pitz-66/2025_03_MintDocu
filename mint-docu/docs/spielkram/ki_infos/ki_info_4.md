Luanti (ehemals Minetest) ist perfekt für leistungsschwache Rechner, da der Server extrem wenig RAM verbraucht. Für eine Gruppe von 2 bis 4 Personen reicht ein gewöhnlicher Office-Laptop völlig aus.
Schritt 1: Installation

   1. Gehen Sie auf die offizielle Webseite luanti.org.
   2. Laden Sie die passende Version für Ihr Betriebssystem (Windows, macOS oder Linux) herunter.
   3. Wichtig: Luanti muss nicht im klassischen Sinne installiert werden. Entpacken Sie die heruntergeladene ZIP-Datei einfach in einen Ordner (z. B. auf dem Desktop).

Schritt 2: Den Server starten (Einfache Methode)
Wenn Sie nur mit Freunden im selben Netzwerk (WLAN) spielen oder eine einfache Verbindung über das Internet herstellen wollen, ist die "Host-Funktion" am einfachsten:

   1. Starten Sie die luanti.exe (im Ordner bin).
   2. Klicken Sie oben auf den Reiter "Spiel beitreten" oder "Einzelspieler".
   3. Wählen Sie eine Welt aus (oder erstellen Sie eine neue).
   4. Setzen Sie auf der rechten Seite den Haken bei "Server hosten".
   5. Geben Sie einen Namen und ein Passwort ein (das Passwort benötigen Ihre Freunde).
   6. Klicken Sie auf "Starten". Ihr Spiel ist nun der Server.

Schritt 3: Freunde einladen
Damit Ihre Freunde beitreten können, benötigen sie Ihre IP-Adresse:

* Im selben Haus (LAN): Drücken Sie Win + R, geben Sie cmd ein und dann ipconfig. Nutzen Sie die "IPv4-Adresse".
* Über das Internet: Sie müssen in Ihrem Router den Port 30000 (UDP) freigeben. Alternativ können Sie Tools wie Tailscale oder Hamachi nutzen, um ein privates Netzwerk ohne Port-Freigabe zu erstellen.

Schritt 4: Den Server anpassen (Kreativmodus)
Damit Sie sofort kreativ bauen können, sollten Sie die Privilegien anpassen:

   1. Drücken Sie im Spiel die Taste T, um den Chat zu öffnen.
   2. Geben Sie /grantme all ein (damit erhalten Sie alle Rechte).
   3. Geben Sie /set -n creative_mode true ein, um unendlich Ressourcen zu haben.

Profi-Tipp: Dedizierter Server (ohne Grafik)
Wenn der Server permanent laufen soll, ohne dass Ihr Spiel offen sein muss:

* Starten Sie im Ordner bin die Datei luantiserver.exe.
* Dieser verbraucht fast keine CPU-Leistung und kann im Hintergrund minimiert bleiben.
